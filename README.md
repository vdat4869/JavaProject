# UTH Conference Management System

## 1. Giới thiệu
UTH Conference Management System là nền tảng hỗ trợ toàn bộ quy trình tổ chức hội nghị khoa học tại Trường Đại học Giao thông Vận tải TP.HCM. Hệ thống cung cấp một quy trình liền mạch từ: Call for Papers, Nộp bài, Phản biện, Quyết định, Nộp camera-ready, Lên chương trình và Xuất kỷ yếu.

**Mục tiêu:**
- Chuẩn hóa quy trình hội nghị.
- Giảm phân tán dữ liệu và thao tác thủ công.
- Tăng tính minh bạch, nhất quán và hiệu quả.
- Tối ưu trải nghiệm cho tác giả, phản biện và ban tổ chức.
